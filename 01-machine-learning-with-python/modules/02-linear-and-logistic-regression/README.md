# Module 2: Linear and Logistic Regression

## What I Learned in This Module

This module covered two fundamental regression techniques for machine learning:
linear and logistic regression. These are fundamental blocks for more advanced
ML algorithms.

I started with understanding what regression actually does (predicting 
outcomes), then worked through simple and multiple linear regression. The cool 
part is that I got to use scikit-learn on real data, not just theoretical 
examples. I also learned about polynomial and non-linear regression for when 
relationships get more complex.

The second part shifted to logistic regression, which is interesting because 
it's actually used for classification (not regression like the name suggests). 
I learned how to train and test classification models properly.

**Bonus:** I got a cheat sheet with all the key concepts, formulas, and use 
cases that I can reference later!
